Downloading from GitHub
-----------------------

Clone the repository:

.. code-block:: bash

  $ git clone git@github.com:starling-lab/starling_theme.git

If necessary: uninstall the theme if a stable release is installed in your environment:

.. code-block:: bash

  $ pip uninstall starling-theme

Change directory to the base of the repository and run ``setup.py``.

.. code-block:: bash

  $ cd starling_theme
  $ python setup.py develop
